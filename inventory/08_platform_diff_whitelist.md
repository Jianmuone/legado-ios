# 平台差异白名单 (Platform Diff Whitelist)

**创建时间**: 2026-04-08

---

## 白名单规则

只有满足以下所有条件的差异才能进入白名单：
1. 该差异由平台机制决定
2. 无法通过原生实现弥补
3. 不改变产品语义
4. 不改变用户操作目标
5. 已给出最接近的等价方案

---

## 白名单条目

### DIFF-001: Content Provider

**Android 能力**: Content Provider 实现跨应用数据共享

**iOS 无法原样复刻的原因**: iOS 不支持 Content Provider 机制

**iOS 等价替代方案**:
- URL Scheme (`legado://import/*`)
- 本地 HTTP API (`http://127.0.0.1:1234/*`)
- App Group (跨应用数据共享)

**用户可感知差异**: 
- 外部应用调用方式不同
- 需要启用 Web 服务

**为什么该差异不构成功能缩水**:
- 所有数据操作接口均已实现
- JSON 格式完全兼容
- 功能语义一致

**是否需要额外提示**: 是，需要在文档中说明 iOS 调用方式

---

### DIFF-002: 后台服务

**Android 能力**: 后台服务长时间运行（下载、缓存、朗读）

**iOS 无法原样复刻的原因**: iOS 后台执行时间限制严格

**iOS 等价替代方案**:
- `BGAppRefreshTask` (后台刷新)
- `BGProcessingTask` (后台处理)
- 音频后台播放 (AVFoundation)
- 用户保持前台运行

**用户可感知差异**:
- 后台任务可能中断
- 需要手动返回应用继续

**为什么该差异不构成功能缩水**:
- 核心功能在前台可用
- 后台朗读已实现
- iOS 平台限制，无法绕过

**是否需要额外提示**: 是，需要在设置中提示后台限制

---

### DIFF-003: 桌面组件

**Android 能力**: 桌面小组件显示书架、阅读进度

**iOS 无法原样复刻的原因**: iOS Widget 功能受限

**iOS 等价替代方案**:
- WidgetKit (需要额外实现)
- 通知提醒

**用户可感知差异**: 暂无桌面小组件

**为什么该差异不构成功能缩水**:
- 非 P0 功能
- 可后续补充 WidgetKit

**是否需要额外提示**: 否

---

### DIFF-004: 广播接收器

**Android 能力**: BroadcastReceiver 监听系统事件

**iOS 无法原样复刻的原因**: iOS 不支持广播机制

**iOS 等价替代方案**:
- `NotificationCenter`
- `Combine` 框架
- `AppDelegate` 生命周期回调

**用户可感知差异**: 无

**为什么该差异不构成功能缩水**: 实现方式不同，功能等价

**是否需要额外提示**: 否

---

### DIFF-005: 文件系统访问

**Android 能力**: 自由访问外部存储

**iOS 无法原样复刻的原因**: iOS 沙盒限制

**iOS 等价替代方案**:
- `UIDocumentPicker`
- `FileManager` (沙盒内)
- `App Group` 共享容器

**用户可感知差异**: 
- 文件导入导出需要用户选择
- 无法直接访问系统目录

**为什么该差异不构成功能缩水**: 
- 所有文件操作均可实现
- 安全性更高

**是否需要额外提示**: 否

---

### DIFF-006: JavaScript 引擎

**Android 能力**: Rhino JS 引擎

**iOS 无法原样复刻的原因**: iOS 使用 JavaScriptCore

**iOS 等价替代方案**: JavaScriptCore

**用户可感知差异**: 
- 部分高级 JS 特性可能不兼容
- `java.*` 类不可用

**为什么该差异不构成功能缩水**:
- 核心 JS 语法完全兼容
- 已实现 JS 桥接函数
- 需要测试回归书源兼容性

**是否需要额外提示**: 否

---

### DIFF-007: WebView 实现

**Android 能力**: WebView 注入 JS 获取内容

**iOS 无法原样复刻的原因**: WKWebView 异步机制

**iOS 等价替代方案**: 
- `WKUserScript` 注入
- `WKScriptMessageHandler` 回调

**用户可感知差异**: 
- WebView 规则执行方式略有不同
- 部分复杂规则可能需要适配

**为什么该差异不构成功能缩水**:
- 核心功能可实现
- 需要针对性测试

**是否需要额外提示**: 否

---

### DIFF-008: 默认存储路径

**Android 能力**: 可自定义存储路径

**iOS 无法原样复刻的原因**: iOS 沙盒限制

**iOS 等价替代方案**: 
- Documents 目录
- App Group 共享目录

**用户可感知差异**: 
- 无法选择存储位置
- 卸载应用数据丢失

**为什么该差异不构成功能缩水**: 
- iOS 平台限制
- 云备份支持

**是否需要额外提示**: 是，需要提示用户定期备份
